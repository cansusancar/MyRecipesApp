


This project is an android recipes application written with Kotlin programming language in Android Studio.
I used these technologies to develop this project:
-MVVM architecture
-Retrofit 
-Dagger-Hilt
-Lottie Animation
-Navigation Drawer
-Battom Navigation Bar
-RecyclerView
